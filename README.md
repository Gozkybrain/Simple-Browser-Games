# Sarcastic Geek - Game Collection

A curated collection of simple arcade games for techies and devs to ease off during breaks. No complex setups, no heavy downloads, just quick games to clear your head between tasks.

## How to use this

1. Open index.html in any browser.
2. Pick a game from the grid.
3. Waste a few minutes of your life.
4. Go back to work (maybe).

## Controls

Most games use standard controls:
- Arrows / WASD for movement.
- Space to start or jump.
- Mouse for pointing and clicking.

## The Games

There are currently 11 games available:
- Snake Game
- Tower Game
- Number Slide
- Sudoku Game
- Chicken Hop
- Memory Cards
- Simon Game
- Sliding Puzzle Game
- Coloron Game
- Hangman Game
